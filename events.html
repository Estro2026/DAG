⚙ Dev Notes — events.html (Fiere ed Eventi) ★★★ Effort medio WordPress Elementor ACF Pro ▾

SEO

  • Title, description, canonical, OG, Twitter Card ✓
  • JSON-LD: WebPage + Event (Première Classe) ✓
  • Creare Assets/og/og-events.jpg (1200×630px)
  • Aggiungere startDate/endDate fiere appena confermati (rich result Google)
  • In Rank Math: tipo pagina "Event", compilare date e location

Elementor / WordPress

  • CPT "Fiere" + ACF (nome, tipo, città, immagine, data) ★★
  • Event Card: Loop Grid + template "Fiera Card" (Inner Section 2 col) ★★
  • Alternativa: The Events Calendar (plugin gratuito) ★★
  • Griglia collaborazioni 3 col: Inner Section, Image + Heading + Text ★
  • Archivio stampa: Media Library + plugin FileBird (cartelle archivio)
  • Contenuto archivio stampa: da completare con la cliente
  • CTA band scura: sezione sfondo #1a1a1a, Heading + 2× Button ★

Assets necessari

  • 1 foto hero 1920×800 (fiera / showroom)
  • 3 foto landscape 4:3 (Première Classe, WHITE, DNA)
  • 3 foto landscape 16:9 (collaborazioni: shooting, capsule, press)
  • 4 foto archivio stampa quadrate
