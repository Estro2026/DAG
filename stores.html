⚙ Dev Notes — stores.html (Store Locator) ★★★★ Effort alto WordPress ACF Pro Google Maps API ▾

SEO

  • Title, description, canonical, OG, Twitter Card ✓
  • JSON-LD: WebPage + Organization ✓
  • Creare Assets/og/og-stores.jpg (1200×630px)
  • In produzione: schema LocalBusiness per ogni rivenditore CPT

Elementor / WordPress

  • Store Locator: usare plugin dedicato (non Elementor nativo)
  • Plugin consigliato: "Store Locator Plus" (gratuito) ★★
  • Alternativa premium: "Agile Store Locator" (UI moderna) ★★★
  • CPT "Rivenditori" + ACF (nome, città, regione, lat/lng) ★★★
  • Google Maps API Key richiesta → Google Cloud Console
  • Maps API: Maps JavaScript + Places API (pay-per-use oltre 28k req/mese)
  • Geolocalizzazione: solo su HTTPS + gestione GDPR cookie
  • js/stores.js è placeholder — sostituire con WP_Query in produzione
  • CTA boutique: Inner Section 2 col, Image + Text Editor ★

Assets necessari

  • Database rivenditori definitivo (nome, indirizzo, città, regione)
  • 1 foto landscape 4:3 (interno boutique)
  • OG image: og-stores.jpg 1200×630px
