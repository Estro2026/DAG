⚙ Dev Notes — shop.html (PLP) ★★★★ Effort alto WooCommerce Elementor ACF ▾

WooCommerce / Elementor

  • Template: WooCommerce archive-product.php override nel tema child
  • Product cards = WooCommerce loop template (content-product.php)
  • Filtri categoria = product attribute taxonomy (pa_tipo) — richiede plugin o custom query
  • Banner made-to-order: ACF global option (Options Page) — campo testo ripetuto in più pagine
  • Avviso tempi di produzione: obbligatorio per legge — Codice del Consumo art. 49
  • Griglia prodotti: WooCommerce loop con woocommerce_product_loop_start/end hooks
  • Ordinamento e filtri: plugin FacetWP o WOOF (WooCommerce Products Filter)

Assets necessari

  • 6 foto prodotto 3:4 — Jaipur, Star, Precious, Holi, Madras, Sun
  • Ogni prodotto: minimo 3 scatti (frontale, dettaglio, indossato)
